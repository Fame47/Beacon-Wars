Beacon Wars v48 GitHub Pages Ready

Why this version exists:
GitHub Pages loads index.html automatically. If your repo only has README.md or a differently named HTML file, the site can show only plain text like a README.

Upload/extract ALL files from this folder into the root of your GitHub repo:
Fame47/Beacon-Wars

The root should contain:
- index.html
- style.css
- script.js
- all PNG/JPG assets
- firebase-config.example.js and notes files

Do not upload only the zip.
Do not leave the game inside a nested folder unless GitHub Pages is pointed to that folder.

GitHub Pages settings:
Settings -> Pages -> Deploy from branch -> main -> /root

Then visit:
https://fame47.github.io/Beacon-Wars/

Hard refresh:
Ctrl + F5

If you still see README text, your index.html is not in the repository root or GitHub Pages has not finished rebuilding.
